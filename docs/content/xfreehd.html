✨ Features Overview

  • ⚡ Fully Asynchronous — High-performance metadata scraping and downloads via asyncio.
  • 🖼️ Album Scraping — Fetch photo gallery structures and automatically extract images by page or all at once.
  • 📥 MP4 Downloader — Built-in support for direct MP4 downloads utilizing multi-threaded segmented range requests via eaf_base_api.
  • ⚙️ Clean Architecture — A lightweight wrapper designed to run cleanly over eaf_base_api.

📦 Installation

Install from PyPI using pip:

bash
pip install unofficial-api-for-xfreehd
Note
Requires Python ≥ 3.12. The networking core package eaf_base_api is automatically fetched and configured.

🚀 Quick Start

All operations are asynchronous and should run within an async context:

python
import asyncio
from xfreehd_api import Client
from base_api import DownloadConfigRAW

async def main():
    client = Client()

    # Fetch a video
    video = await client.get_video("https://xfreehd.com/video/12345/example-video")

    # Access metadata
    print(video.title)
    print(video.views)

    # Download video using RAW downloader (direct MP4)
    config = DownloadConfigRAW(quality="hd", path="./downloads")
    await video.download(configuration=config)

    # Fetch an album and get image links
    album = await client.get_album("https://xfreehd.com/album/12345/example-album")
    images = await album.get_all_images()
    print(f"Found {len(images)} images in album.")

asyncio.run(main())

⚙️ Configuration

XFreeHD API uses eaf_base_api for networking. You can configure proxies, timeouts, limits, and HTTP parameters via a custom BaseCore.

Please refer to the eaf_base_api Documentation for the complete reference on how to set up RuntimeConfig and BaseCore.

python
from base_api import BaseCore
from base_api.modules.config import RuntimeConfig
from xfreehd_api import Client

my_config = RuntimeConfig()
my_config.proxies = {"https": "socks5://127.0.0.1:9050"}

core = BaseCore(configuration=my_config)
client = Client(core=core)

⬇️ Downloading Options

XFreeHD video files are direct MP4 assets on CDNs. Configure downloads using DownloadConfigRAW:

python
from base_api import DownloadConfigRAW

config = DownloadConfigRAW(
    quality="hd",             # Set to "hd" for high quality, or "sd" for standard quality
    path="./downloads",        # Destination path
    no_title=False,            # If False, automatically appends title + ".mp4"
    allow_multipart=True,      # Enable parallel segmented downloads using range requests
    max_workers=5              # Concurrent download segment threads
)

success = await video.download(configuration=config)

For full details on RAW download parameters, see the reference in eaf_base_api Documentation.

📊 Scraping Results

Iterating methods like client.search() return async generators yielding ScrapeResult instances:

python
async for result in client.search("beach", pages=2):
    if result.is_success:
        video = result.video        # The Video object
        print(video.title)
    else:
        print(f"Error for {result.url}: {result.error}")

ScrapeResult Attributes

AttributeTypeDescription
urlstrThe parsed video target URL
videoVideo | NoneThe parsed Video object (if successful)
errorException | NoneThe exception raised (if failed)
is_successboolWhether the scraping task succeeded

⚠️ Error Handling

Custom exceptions are defined in xfreehd_api.modules.errors. Network-level errors bubble up from the underlying eaf_base_api package.

ExceptionTrigger Cause
NotFoundServer returned HTTP 404 (e.g. video/album deleted)
NetworkErrorRequest failed due to HTTP connection problems
BotDetectionCloudflare challenge block detected
ProxyErrorProxy configuration failed or proxy is down
UnknownNetworkErrorUnexpected network errors
DownloadFailedRaw segmented download failed
python
from xfreehd_api.modules.errors import NotFound, BotDetection

try:
    video = await client.get_video(url)
except NotFound:
    print("This video does not exist!")
except BotDetection:
    print("Scraper was blocked by anti-bot measures.")
