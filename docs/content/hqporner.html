📦 Installation

Install from PyPI using pip:

bash
pip install hqporner_api

For CLI formatting support (recommended), install with the optional cli dependency:

bash
pip install hqporner_api[cli]
Note
Requires Python ≥ 3.12. The underlying networking library eaf_base_api is installed automatically as a dependency.

🚀 Quick Start

Every method in this API is asynchronous. You need to run your code inside an async function:

python
import asyncio
from hqporner_api import Client

async def main():
    client = Client()

    # Fetch a video
    video = await client.get_video("https://hqporner.com/hd/...")

    # Access metadata
    print(video.title)
    print(video.length)
    print(video.pornstars)

    # Download the video
    from hqporner_api import DownloadConfigRAW
    config = DownloadConfigRAW(quality="best", path="./downloads")
    await video.download(configuration=config)

asyncio.run(main())

⚙️ Configuration

The entire API relies on eaf_base_api for its networking. You can configure global settings (proxies, timeouts, etc.) via the singleton config, or pass a custom BaseCore into the Client.

Please refer to the eaf_base_api Documentation for the complete reference on how to set up RuntimeConfig and properly integrate it with this API.

python
from base_api import BaseCore
from base_api.modules.config import RuntimeConfig
from hqporner_api import Client

my_config = RuntimeConfig()
my_config.proxies = {"https": "socks5://127.0.0.1:9050"}

core = BaseCore(configuration=my_config)
client = Client(core=core)

⬇️ Downloading

HQPorner videos host direct MP4 media files instead of HLS streams. All downloads are configured using the DownloadConfigRAW dataclass:

python
from hqporner_api import DownloadConfigRAW

config = DownloadConfigRAW(
    quality="best",           # "best", "half", "worst", or height int (e.g. 720)
    path="./downloads",       # Output directory or file path
    no_title=False,           # Auto-append video.title + ".mp4" if False
    allow_multipart=True,     # Use multi-threaded range requests
    max_workers=5             # Number of parallel download threads
)

success = await video.download(configuration=config)

📄 Pagination & Iterators

Methods like search_videos(), get_videos_by_actress(), and get_videos_by_category() return async generators yielding ScrapeResult instances:

python
async for result in client.search_videos("college", pages=2):
    if result.is_success:
        video = result.video
        print(video.title)
    else:
        print(f"Failed URL {result.url}: {result.error}")

ScrapeResult

AttributeTypeDescription
urlstrThe video URL
videoVideo | NoneThe populated Video object (if successful)
errorException | NoneThe error that occurred (if any)
is_successboolWhether the scrape succeeded

Concurrency Parameters

All iterator methods accept concurrency tuning:

  • videos_concurrency int | None — Max videos loaded concurrently. Defaults to config.videos_concurrency (5).
  • pages_concurrency int | None — Max pages fetched concurrently. Defaults to config.pages_concurrency (2).
  • on_video_error (url, error, attempt) → bool — Custom error callback. Return True to retry.
  • on_page_error (url, error, attempt) → bool — Same but for page fetch failures.
  • keep_original_order bool — Yield results in page order (slower but deterministic).

⚠️ Error Handling

Custom exceptions are defined in hqporner_api.modules.errors. Base networking/session exceptions are raised from eaf_base_api.

ExceptionWhen Raised
InvalidActressThe actress name or URL pattern did not match checks

BotDetectionCloudflare or similar bot protection triggered
ProxyErrorInvalid or failing proxy
UnknownNetworkErrorUnexpected network errors

python
from hqporner_api.modules.errors import NotFound, BotDetection

try:
    video = await client.get_video(url)
except NotFound:
    print("Video does not exist")
except BotDetection:
    print("Bypassing bot protection failed")

💻 CLI Usage

The HQPorner API package includes a rich CLI:

bash
# Download a single video
hqporner_api --download "https://hqporner.com/hd/..." --quality best --output ./downloads --no-title False

# Search and download resulting videos
hqporner_api --search "blonde" --pages 2 --quality 1080 --output ./blonde_videos/ --concurrency 3

CLI Options
